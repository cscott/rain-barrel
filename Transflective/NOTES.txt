1. Hack LCD tester board:
   a. bodge in 4.7uF caps for C8/C10/C11 (and then for C2/C3/C4 if
      needed)
   b. Check voltage drop across R1, maybe drop to 100 ohm
   c. Hack interface to LCD Panel board: 7x2
      pin 1 - 3.3V
          3 - RST
          4 - RST (see point 2 below)
          5 - SCL
          7 - SI
          9 - XCS
          11 - 5V
          13 - GND
2. /RST routing -- need an independent source of RST toggle for LCD
   Panel. Some options:
   a. Bodge LCD Panel board to bridge TP9 (pin 4 on interboard header)
      with DS_RST (a little tricky, might have to solder to FPC header;
      this is easier if we break the /RST trace on the rear panel
      board so we can just bodge between pin 3 and 4 on the interboard
      header)
   b. Bodge Rear Panel board to bridge pin 3 on interboard header to
      <unknown source of additional output signal>
      (as before, maybe easiest just to break existing pin 3 trace,
      short pin 3 and 4, then use the existing pad for pin 4)
   c. Bodge LCD panel board to bridge DS_RST with <future GPIO>,
      maybe one from standalone current regulator board
      (this is easier if we break the trace on the main board so
      we can solder to pin 3 on interboard header, instead of finding
      the signal on the FPC header)
   d. If we don't need DS_CS, repurpose that.  Again, probably easier
      if we break the /RST trace on the rear panel, and then bridge
      pins 3 and 9 of the interboard header on either the rear panel
      board or the lcd panel board.
3. Do we need LCD_CS -- experiment w/ the LCD tester board and
   see if we can leave DS_CS grounded all the time.
4. LCD panel board: the 1uF caps on the LCD interface need to be
   35V caps, the boost voltage is easily ~25V (although we're trying
   to manage that)
5. Controlling backlight: design standalone ADP1650 board that plugs
   in via STEMMA QT (ie, 3.3v power) and connects to LCD backlight tabs

6. Rewire J17 to F1
7. Rewire pump output to F5 (verify that we can make this independent
   of SPI); use F2 for RST sigal
